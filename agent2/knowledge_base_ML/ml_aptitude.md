# ML Math, Stats & Aptitude

## Probability & Statistics
* Foundations: Bayes' Theorem, Conditional Probability, Independence, Variance.
* Distributions: Normal, Binomial, Poisson, Bernoulli, Uniform.
* Statistical Inference: Hypothesis testing (A/B testing), p-values, Confidence Intervals, Central Limit Theorem (CLT).
* ML Stats: Maximum Likelihood Estimation (MLE), Bias-Variance Tradeoff, Covariance vs. Correlation.

## Linear Algebra
* Core: Vectors, Matrices, Matrix Multiplication, Dot Product.
* Advanced: Eigenvalues, Singular Value Decomposition (SVD), Principal Component Analysis (PCA).

## Calculus & Optimization
* Calculus: Derivatives, Partial Derivatives, Chain Rule (Backpropagation).
* Optimization: Gradient Descent, Convex vs. Non-Convex functions, Local Minima.
* Regularization: Difference between L1 (Lasso) and L2 (Ridge) regularization.

## ML Fundamentals (Metrics)
* Evaluation Metrics: Accuracy, Precision, Recall, F1-Score, ROC-AUC.
* Concepts: Overfitting/Underfitting (identifying and fixing via Dropout/Early Stopping).
